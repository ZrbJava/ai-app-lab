# AI 应用开发术语表

学习过程中遇到的概念在此记录，方便复习。

| 术语 | 一句话解释 | 相关 Lab |
|------|-----------|----------|
| **LLM** | 大语言模型，根据文本预测下一个 token | 02 |
| **Prompt** | 发给模型的输入文本 / 消息列表 | 01 |
| **SSE** | Server-Sent Events，服务端单向推流到浏览器 | 02 |
| **Streaming** | 模型逐 token 输出，非一次性返回 | 02 |
| **Provider** | 模型供应商（OpenAI、Anthropic、Ollama 等） | 03 |
| **BYOK** | Bring Your Own Key，用户自带 API Key | 06 |
| **Tool / Function Calling** | 模型决定调用外部函数并拿结果继续推理 | 04 |
| **Agent** | 能多步推理、使用工具的 LLM 系统 | 07 |
| **RAG** | 检索增强生成：先查文档再回答 | 09 |
| **Embedding** | 文本的向量表示，用于相似度检索 | 09 |
| **Artifact** | Agent 中间产物 UI（思考、任务、结果等） | 08 |
| **Session** | 一次对话线程 | 05 |
| **Async Job** | 长耗时任务（视频生成）异步执行 | 10 |
